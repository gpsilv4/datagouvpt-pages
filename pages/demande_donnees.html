Suggérer une ouverture de données

Vous pouvez suggérer la publication d’un jeu de données qui n’est pas actuellement disponible sur data.gouv.fr. Ces demandes aident notre équipe à identifier les besoins et prioriser les efforts d’ouverture des données en collaboration avec les producteurs de données.

Faire une demande

Vous pouvez explorer les demandes formulées ainsi que voir le jeu de données

N’hésitez pas à utiliser l’espace commentaire du jeu de données pour échanger sur les demandes.

Pour toutes autres demandes, vous pouvez nous écrire.

NOTE : Les demandes effectuées ici ne sont pas assimilables à une demande adressée à la Commission d’Accès aux Documents Administratifs.
